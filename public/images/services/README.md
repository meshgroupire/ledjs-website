# Service Images Folder

Place your service images in this folder and update the references in `src/config/services.js`

## Image Guidelines:
- **Format**: JPG or PNG
- **Dimensions**: Recommended 800x500px or 1200x600px (rectangular format)
- **File naming**: Use the service ID from services.js (e.g., `wedding-djs.jpg`, `party-djs.jpg`)

## Current Service Images Required:

- party-djs.jpg
- led-wristbands.jpg
- speed-quizzing.jpg
- music-bingo.jpg
- karaoke.jpg

## To Update Images:
1. Add your new image file to this folder with the correct filename
2. The image will automatically appear on the website
3. No code changes needed!

## Placeholder Images:
If you don't have images yet, placeholder images will be shown automatically. Replace them with your actual images when ready.
